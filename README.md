# eecs351project
Website and source files for EECS 351 final project

# Demo Instructions
1. Clone this repo (specifically master branch)
2. Navigate into the repo folder in MATLAB
3. Type `driver`

* To test with K^{th} Nearest Neighbors instead of Mahalanobis, uncomment lines
  64-65 and comment lines 60-61
* To test with envelope features in addition to MFCC features, uncomment lines
  28, 36, 44
